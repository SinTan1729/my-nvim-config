# My NeoVim Config Files

This is my config for neovim. It's a bit over-engineered I guess, but hey, it fits my needs.
There are two modes for this config. The default `base` one that's used by default, and then an
`extra` layer that's activated by setting the environment variable `NVIM_CONFIG_MODE` to `full`.

Feel free to use it as a baseline for your personal [NeoVim](https://neovim.io) configuration.
