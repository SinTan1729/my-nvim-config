# My NeoVim Config Files

This is basically a collection of copies of `$XDG_CONFIG_HOME/nvim` from all my
machines.

The `full` one is for my work machine, and the `minimal` one is for all other machines where I
don't really do any coding.

Feel free to use it as a baseline for your personal [NeoVim](https://neovim.io) configuration.
